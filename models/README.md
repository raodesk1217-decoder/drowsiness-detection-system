# Model File Required

This project requires the pre-trained Dlib model:

shape_predictor_68_face_landmarks.dat

Due to GitHub file size restrictions and licensing, the model file is not included in this repository.

## Download Instructions

1. Download from:
http://dlib.net/files/shape_predictor_68_face_landmarks.dat.bz2

2. Extract the file.

3. Place the extracted file inside this folder:

models/

Final structure:

models/
└── shape_predictor_68_face_landmarks.dat
